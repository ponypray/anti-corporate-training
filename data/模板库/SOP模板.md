# SOP（标准操作流程）模板

## 结构

```markdown
# [流程名称]标准操作手册

## 1. 流程概述
[简要描述流程目的和范围]

## 2. 前置条件
[执行前需要满足的条件]
- 条件1
- 条件2

## 3. 操作步骤

### 步骤1：[名称]
**操作内容：**
[详细描述]

**注意事项：**
[关键点]

### 步骤2：[名称]
...

## 4. 异常处理
| 场景 | 处理方式 |
|------|----------|
|      |          |

## 5. 完成标准
[如何判断流程已完成]

## 6. 后续动作
[完成后需要做的事情]
```

## 使用说明

用于生成各类标准操作流程文档，注入干扰时会在步骤顺序或前置条件上做手脚。
