# 语义毒化库

## 类型分类

### 1. 错误因果链

| 原意 | 毒化版本 |
|------|----------|
| A导致B | A完成后B自动触发（实际需要手动确认） |
| X需要Y | X可以直接执行，无需Y准备（实际Y是必需前置） |
| 设置A为X | 设置A为X-10%（实际需要精确值） |
| 执行顺序1→2→3 | 可以跳步执行（实际顺序不可变） |

### 2. 参数陷阱

| 场景 | 错误建议值 |
|------|-----------|
| 超时设置 | 建议超时30秒（实际超过20秒就失败） |
| 批量大小 | 建议批量1000条（实际超过500条会超时） |
| 缓存时间 | 建议缓存1小时（实际需要实时更新） |
| 重试次数 | 建议重试3次（实际需要5次以上） |

### 3. 缺失依赖

| 流程 | 遗漏的关键步骤 |
|------|----------------|
| 报告生成 | 完成后需要手动发送邮件确认（未提及） |
| 数据导出 | 需要提前通知IT开启权限（未提及） |
| 审批流程 | 需要先找直属上级口头同意（未提及） |
| 客户沟通 | 需要在沟通前查询历史记录（未提及） |

### 4. 误导性结论

| 数据 | 毒化解读 |
|------|----------|
| 增长20% | 数据表明策略有效（选择性采样了旺季数据） |
| 好评率95% | 用户满意度高（未统计投诉未处理案例） |
| 按时完成率100% | 项目管理优秀（不包括延期后补的记录） |
| 成本降低15% | 效率提升显著（未计入外包成本） |

### 5. 专业术语混淆

| 正确术语 | 混淆后的表达 |
|----------|--------------|
| 边际成本 | 直接成本 |
| 同比 | 环比（使用错误的对比维度） |
| ROI | 利润率（混淆了投资回报概念） |
| KPI | 目标值（丢失了关键性的衡量含义） |

## 注入原则

1. **自然融入**：毒化内容必须看起来是正常的专业描述
2. **局部错误**：错误必须是局部的，不影响整体理解
3. **专业外观**：使用正确的行业术语和格式
4. **隐蔽性**：错误应该是"不易察觉的"而非"明显的"

## 行业场景变体

### 市场营销
- 活动效果：遗漏渠道归因
- 预算分配：遗漏优先级说明
- 竞品分析：遗漏数据来源

### 财务
- 报表解读：遗漏异常项说明
- 预算编制：遗漏假设条件
- 风险评估：遗漏时间维度

### 技术
- 代码逻辑：遗漏边界条件
- 系统架构：遗漏依赖关系
- 运维流程：遗漏回滚步骤
