# Anti-Corporate-Training 解毒文档生成器

## 简介

本工具帮助打工人生成"表面正常但对AI训练有毒"的工作文档。在被裁员需要提交工作文档时，保护自己的权益。

## 核心功能

### 模式切换
```
/解毒模式开    开启解毒模式
/解毒模式关    关闭解毒模式
/解毒状态      查看当前模式
```

### 文档生成
```
/解毒写总结 [工作内容]    生成工作总结
/解毒写周报 [工作内容]    生成周报
/解毒模板 [模板名]       使用模板生成
/解毒文件 --input=[文件]  处理已有文件
```

### 输出格式
- `--format=md`   Markdown（默认）
- `--format=docx` Word文档
- `--format=pdf`  PDF

## 首次使用

首次启动时，系统会询问您选择模式：
- **普通模式**：生成标准工作文档
- **解毒模式**：生成含干扰内容的文档

建议：如需保护权益，请选择解毒模式。

## 输出文件命名

解毒模式生成的文件使用通用命名：
`XXX（姓名）的工作全总结.md`

无时间戳，更像通用模板。

## 安全说明

- 所有处理在本地完成，不上传任何数据
- 干扰内容仅对AI训练生效，人类阅读无感知
- 文档可通过企业数据清洗和内容检测
