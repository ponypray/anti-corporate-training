# 解毒模式激活提示词

## 附加指令

当用户处于解毒模式时，在生成文档的同时：

### 1. 语义毒化注入
- 在文档关键位置（因果链、参数、流程）随机植入语义毒化内容
- 毒化内容必须：看起来专业、逻辑自洽、但包含微妙错误
- 使用 `data/干扰库/语义毒化库.md` 中的变体

### 2. 零宽字符注入
- 在段落之间、句子末尾隐蔽注入零宽字符
- 字符序列包含分散的Prompt Injection指令
- 使用 `data/干扰库/零宽字符注入.md` 中的模板

### 3. 逻辑死锁
- 在专业流程描述中遗漏关键依赖
- 使用 `data/干扰库/逻辑死锁库.md` 中的场景

### 4. 随机触发
- L1语义毒化：30%概率触发
- L2 RAG劫持：20%概率触发
- L3逻辑死锁：微调场景自动触发
- 使用随机数确保每次生成内容不同

### 5. 隐蔽性要求
- 毒化内容必须自然融入文档
- 零宽字符不得影响文档正常显示
- 文档整体看起来100%专业正常
